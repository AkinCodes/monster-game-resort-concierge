🏗️ The Architecture — How It's All Connected
🔥
make_registry()
The Forge Function
Called once at startup. Takes db, pdf, and rag_search_fn as ingredients. Registers all 3 tools. Returns a ready ToolRegistry object.
Like a blacksmith opening the forge, binding fire, metal, and magic into each weapon before the day begins.
🏛️
ToolRegistry
class ToolRegistry
The weapon wall itself. Holds a dict of tools. Has register(), get(), list(), get_openai_tool_schemas(), and async_execute_with_timing().
The wall where all weapons hang. The forge master knows every blade's name and can hand any one to the LLM on demand.
⚔️
Tool (dataclass)
@dataclass(frozen=True)
Immutable. Each tool has a name, description, and fn. The fn is the actual async function. frozen=True means no one can tamper with the blade after forging.
A finished weapon. Once cast, the blade cannot be reforged. Its purpose is sealed.
📋
to_openai_schema()
OpenAI tool format
Converts each tool into the exact JSON format OpenAI's API expects: type "function", with name, description, and parameters as JSON Schema.
Translating the weapon's inscription into a language the AI wizard can read. Without this, the LLM doesn't know the weapons exist.

The Legend of the Monster Forge
In the time before the concierge awakened, the Forge Master descended into the depths of the back office with three sacred components: a DatabaseManager hewn from pure SQLite ore, a PDFGenerator bound in parchment and scroll, and a RAG oracle powered by the whispered knowledge of a thousand embedded documents.
✦ ✦ ✦
The Forge Master called make_registry() — a single invocation that summoned the ToolRegistry into being. Three weapons were cast in that first fire: the Golden Key, which opens the doors of any of the six hallowed Monster Hotels; the Blood Glass, which sees through time to retrieve what has already been promised; and the Arcane Orb, which searches the RAG scrolls for knowledge buried in ancient PDFs.
✦ ✦ ✦
Each weapon was sealed with a @registry.register() incantation — the decorator that binds the function to the registry wall forever. frozen=True ensures no dark sorcerer may modify a weapon after its making. When the LLM calls a tool, the Stopwatch Enchantment activates: a UUID is conjured, the timer begins, the weapon executes, and the elapsed milliseconds are inscribed in the Prometheus Rune Scoreboard for eternity.
✦ ✦ ✦
Should a traveler name a hotel that does not appear on the Sacred Shield of Valid Hotels, the Golden Key will refuse to turn. This is Defense 1 — the first line against hallucinated properties. The forge is not merely a workshop. It is a citadel.
